## ✨ what is santi.uninstaller?

**santi.uninstaller** is a modern Windows application designed to remove installed programs and clean up the files they leave behind.

Unlike the standard Windows uninstaller, santi.uninstaller scans your system for installed applications, runs their official uninstallers, and searches for leftover files and registry entries afterward.

---

## 🚀 what does it do?

* 🔍 scans the Windows registry for installed programs
* 🗑️ runs each program's official uninstaller
* 🧹 finds leftover files, folders, and registry entries
* ⚡ supports forced removal for stubborn programs
* 🛡️ retries uninstallers with administrator permissions when needed
* 📦 allows multiple programs to be selected at once
* 🔎 includes program searching and sorting
* 🎨 provides a clean and modern interface
